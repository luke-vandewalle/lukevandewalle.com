Arlington BirdNET

A data dashboard providing a live overview of bird calls near my apartment, featuring an interactive map that displays national bird hearing data from eBird. The back end is powered by Python using Flask, matplotlib, and SQLite3, and runs on a Raspberry Pi Zero 2 W using BirdNET for audio analysis. The front end, hosted on a Raspberry Pi 4, is built with HTML, CSS, and JavaScript to deliver a responsive and interactive user experience.
